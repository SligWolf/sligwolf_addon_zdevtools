# SligWolf's ZDevTools

This is an internal development and testing tool for SligWolf's Addons. <br/>
Do not use on production servers/clients. Only use this during addon development. <br/>
Use this addon at your own risk! This will not be publicly available on Workshop! <br/>
The name makes sure it appears last in lists and load orders. <br/>

Do not reupload in any way! <br/>

Safe operation:
 - Convar: `dev_sligwolf_zdevtools_developers_enabled 0` (default)
